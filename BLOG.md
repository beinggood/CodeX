# 用 Tkinter 体验 RVO2 多智能体避障：一个可交互的 Python 小项目

如果你正在学习多智能体避障（RVO2 / ORCA 等相关概念），这个项目值得你试一试。它用 **Python + Tkinter** 实现了一个轻量可交互的可视化演示：让多个智能体朝着目标移动，同时自动避开彼此与障碍物，直观呈现“群体避障”的核心思想。

---

## 为什么推荐这个项目？

学习多智能体避障时，经常会遇到两个问题：

1. **概念很抽象**：数学公式与论文很难与直觉对应；
2. **缺少可视化反馈**：看不到智能体的速度方向与避障效果。

这个项目专门解决这两个痛点。你可以直接运行 GUI，**观察每个智能体的实时速度箭头**，并通过点击画布手动添加智能体或障碍物，快速验证自己的理解。

---

## 项目亮点

- ✅ **实时多智能体避障演示**  
  自动模拟多个智能体向目标移动时的相互避障行为。

- ✅ **可视化速度方向**  
  每个智能体都有清晰的速度箭头，还可以一键隐藏/显示。

- ✅ **交互式场景搭建**  
  在画布上点击即可添加智能体或障碍物，无需修改代码。

- ✅ **轻量、易运行**  
  只需 Python 3，自带 Tkinter，无需额外依赖。

---

## 适合谁？

这个项目非常适合：

- 对 **机器人避障 / 多智能体** 感兴趣的同学
- 正在学习 **RVO2 / ORCA** 的研究者或工程师
- 想快速做教学演示、课堂展示的老师
- 想用 Python 做可视化练手的小伙伴

---

## 快速开始

确保本地已安装 Python 3，然后运行：

```bash
python main.py
```

---

## 交互操作说明

运行后你会看到一个 GUI 窗口，可操作的功能包括：

- **开始 / 暂停**：控制模拟运行
- **重置**：重建场景并重新生成智能体
- **添加Agent**：点击画布添加新的智能体
- **添加障碍物**：点击画布添加圆形障碍物
- **取消添加**：退出添加模式
- **显示速度**：显示/隐藏速度方向箭头
