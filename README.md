# RVO2 避障演示（Python GUI）

这是一个使用 **Tkinter** 实现的 RVO2 风格多智能体避障可视化演示。
应用支持在二维画布中展示多个智能体朝目标移动时的相互避障行为，并提供手动添加智能体与障碍物的交互功能。

## 功能特性

- 多智能体朝目标移动并相互避障
- 可视化显示每个智能体的当前速度方向箭头
- 支持切换是否显示速度箭头
- 支持点击画布手动添加智能体或障碍物
- 一键重置场景

## 运行方式

确保本地安装了 Python 3，直接运行：

```bash
python main.py
```

## 交互说明

- **开始 / 暂停**：控制模拟运行与暂停
- **重置**：重建场景并随机生成智能体与目标
- **添加Agent**：点击画布添加智能体
- **添加障碍物**：点击画布添加圆形障碍物
- **取消添加**：退出添加模式
- **显示速度**：切换显示/隐藏速度箭头

## 备注

该演示实现了简化的 RVO2 思路用于教学与可视化，便于快速理解多智能体避障的基本概念。
